# RCS-0031 shared-contract-package-trigger

## Code

- `RCS-0031`

## Goal

Evaluate whether repeated stable contracts justify introducing first shared package under `packages/`.

## Acceptance

- [ ] Stability criteria are checked against at least two repeated slices.
- [ ] Decision to defer or implement shared package is documented.
- [ ] If implemented, migration plan is scoped to one contract family.

## Notes

Do not extract shared packages prematurely.
